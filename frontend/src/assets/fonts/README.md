# 메이플스토리 전용 폰트

이 폴더에 메이플스토리 전용 폰트 파일을 넣어주세요.

- 공식 다운로드: https://maplestory.nexon.com/Media/Font
- 필요한 파일 예시: `Maplestory Light.ttf`, `Maplestory Bold.ttf`
  (실제 배포 파일명이 다르면 `src/styles/global.css`의 `@font-face` `src` 경로를 맞춰 수정하세요)

## 라이선스 안내
넥슨 메이플스토리 폰트는 무료로 배포되지만 자체 라이선스 조항이 있습니다.
상업적 이용 범위 등 배포 페이지의 라이선스 조건을 사용 전 꼭 확인하세요.
